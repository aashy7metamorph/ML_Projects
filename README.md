# Recommendation System

## Overview
Movie recommendation system that suggests items based on user behavior and item attributes.

## Features
- User-based & Item-based Collaborative Filtering  
- Content-Based Filtering  
- Hybrid Approach combining both methods  
- Evaluation with RMSE, Precision@K, Recall@K  

## Tech Stack
- Python, Pandas, NumPy, Scikit-learn, Surprise  
- Optional: Streamlit/Flask for interactive interface  


## Dataset
- Any user-item-rating dataset (e.g., MovieLens)

## Future Work
- Deep learning-based models  
- Real-time recommendations  
- Improved hybrid model performance
